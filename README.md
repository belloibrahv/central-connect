# CentralConnect (Student Accommond8)

## Quick start
1. Install dependencies.
2. Copy `.env.example` to `.env` and fill in keys.
3. Run database migrations and seed data.
4. Start the dev server.

## Commands
- `npm install`
- `npm run prisma:migrate`
- `npm run db:seed`
- `npm run dev`

## Notes
- The seed creates the 7 hostels, default room types, and one open full-session term.
- Add rooms and beds in the admin panel (to be built next).
